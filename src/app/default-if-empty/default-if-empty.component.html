defaultIfEmpty(缺省) 说明

签名: defaultIfEmpty(defaultValue: any): Observable

如果源 Observable 在完成之前没有发出任何 next 值，则发出给定的值，否则返回 Observable 的镜像

如果源Observable本来就是空的,那么这个操作符会发出一个默认值

弹珠图解析:
- 上游操作符没有发出任何值,且发出已完成信号
- 则输出缺省值 42
