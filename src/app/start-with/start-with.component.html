对比分析
startWith(前缀序列) 说明

签名: startWith(an: Values): Observable

发出给定的第一个值(可以设置多个值)

返回的 Observable 会先发出作为参数指定的项，然后再发出由源 Observable 所发出的项

你希望修改过的 Observable 可以先发出的项

BehaviorSubject 也可以从初始值开始！

注意,如果使用 TS 编写代码, 则会检查输出值的类型是否与源 Observable 的值类型一致

弹珠图解析:
- 源 Observable 的序列为 `a` `b` `c`
- startWith 操作符, 在订阅源 Observable 前先输出值 `s`
